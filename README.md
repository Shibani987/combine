# Weekend Challenge - 3 Projects in 1 Webpage

This is a Day 7 weekend challenge project where three previous mini projects are combined into one responsive webpage.

## Projects Combined

- Digital Clock
- Tip Calculator
- BMI Calculator

## Features

- Live animated digital clock using HTML Canvas
- Tip calculator with custom tip percentage and people split
- BMI calculator with metric and imperial unit support
- Responsive layout for desktop and mobile screens
- Clean card-based dashboard design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

Open `index.html` in any modern web browser.

## Folder Structure

```text
combine/
  index.html
  styles.css
  script.js
  README.md
```

## Purpose

The purpose of this project is to practice combining multiple JavaScript projects into one single webpage while keeping the layout organized and user-friendly.
